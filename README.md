# mat-engine-dashboard

Cockpit-inspired dashboard engine dengan modular UI dan emotional feedback.

## 🚀 Features
- Signal feed untuk real-time data
- Error catcher dengan log narrative
- Animation hooks untuk resonant feedback
- Modular components senang extend

## 📂 Project Structure

mat-engine-dashboard/
├── src/
│   ├── components/    # UI instruments (gauges, panels, alerts)
│   ├── hooks/         # Custom React hooks (animation, feedback)
│   ├── store/         # State management (context/redux)
│   └── utils/         # Helper functions
├── public/            # Static assets
├── package.json
└── README.md
